# AGENTS.md — northyorkshirepsychotherapy.com

Single-page static site for Clare Urmston's psychodynamic psychotherapy
practice. Deployed via Netlify, DNS through Cloudflare.

## Stack

Static HTML/CSS, no build step. Copy lives in `index.html`; styles in
`assets/css/style.css`.

## Real-world content

The site carries real practitioner details (name, address, phone, email).
Treat changes to those as deliberate edits, not refactors — and never
auto-fill placeholder data.

## Source documents

`docs/` is gitignored on purpose — client-prep notes and drafts stay
out of the public repo.
